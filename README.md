**在《杀戮尖塔2》多人联机中，用表情包与队友即时互动！**  
点击按钮，从你自己的图片库中选择表情，图片会以动画气泡的形式出现在你的角色头顶，并同步给所有队友。

## ✨ 主要功能

- **📁 便捷管理**：主菜单左下角新增“管理表情包”和“导入图片”按钮，一键打开文件夹或选择本地图片，自动缩放并转为 PNG 格式。
- **😄 战斗即时发送**：战斗界面右下角（可自定义位置）新增“表情”按钮，点击后弹出表情选择面板，选择即发送。
- **🌐 联机同步**：表情通过网络实时广播给所有队友，对方角色头顶会显示相同的动画气泡。
- **🎨 自动适配**：发送的图片自动等比例缩放（最大边长 256 像素），气泡显示时保持原始宽高比，不变形。
- **✨ 流畅动画**：表情气泡带有淡入、悬停、上浮淡出的平滑动画效果。

---

## 🚀 安装与使用

1. **下载模组**  
   从 [Releases](https://github.com/yuyingxingji/slay-the-spire2-emjoy/releases) 页面下载最新版本的压缩包。

2. **解压到游戏目录**  
   将解压得到的 `slay the spire2-emjoy` 文件夹放入游戏的 `mods` 文件夹中。
