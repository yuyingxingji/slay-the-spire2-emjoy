## 🚀 安装与使用

1. **下载模组**  
   从 [Releases](https://github.com/yuyingxingji/slay-the-spire2-emjoy/releases) 页面下载最新版本的压缩包。

2. **解压到游戏目录**  
   将解压得到的 `slay the spire2-emjoy` 文件夹放入游戏的 `mods` 文件夹中。
   
3. **导入表情**  
   选择想要导入的表情(暂不支持.gif文件)。
